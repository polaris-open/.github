# .github

This repository holds the public profile and the default community health files for the [Polaris](https://github.com/polaris-open) organization.

GitHub applies these files across all Polaris repositories that don't define their own, so they act as organization-wide defaults.

## What's here

- [`profile/README.md`](profile/README.md) — the public profile shown on the [organization page](https://github.com/polaris-open).
- [`CONTRIBUTING.md`](CONTRIBUTING.md), [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md), [`GOVERNANCE.md`](GOVERNANCE.md) — how to contribute and how the community is run.
- [`SECURITY.md`](SECURITY.md), [`SUPPORT.md`](SUPPORT.md) — reporting vulnerabilities and getting help.
- [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) and [`.github/pull_request_template.md`](.github/pull_request_template.md) — default issue and pull request templates.

This repository is not the Polaris website and not Polaris Academy. It is intentionally small and institutional.
