# Polaris

> Open knowledge, built by the community, for those building the future.

Polaris is an open source community for people who want to learn, build, and share serious work in:

- Software engineering
- Applied AI
- Open source
- Technical growth

We are starting small, in public, with focus on:

- Self-paced learning
- Project-based practice
- Open source collaboration
- Engineering depth
- Critical thinking
- Responsible use of AI
- Practical artifacts over hype

---

## Current initiatives

**[Polaris Academy](https://github.com/polaris-open/polaris-academy)** — Open, self-paced learning paths for software engineering, applied AI engineering, and technical growth.

**[Runa](https://github.com/polaris-open/runa)** — Local-first, Markdown-first agent for knowledge bases and project workflows.

**[Agentic Squad](https://github.com/polaris-open/agentic-squad)** — A framework for structuring agentic software engineering workflows across teams, repositories, and SDLC stages.

---

## Status

Polaris is in an early draft stage.

Some repositories may contain incomplete ideas, early architecture, experiments, or starter materials. We prefer to be clear about maturity instead of pretending everything is production-ready.

---

## Principles

- Open knowledge, not gatekeeping.
- Learning by building.
- Community as support, not dependency.
- Serious engineering over hype.
- Responsible AI by default.
- Synthetic data only in public learning projects.
- Clear boundaries: demos are not production.
